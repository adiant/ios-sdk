AdbladeErrorCodes

public enum AdbladeErrorCodes: Int

Different error codes.

  • UnknownError
  • ParsingError An error parsing the response from the ad server.
  • RequestError An error with the ad server request.
  • InvalidAdTypeError
  • NoFill
  • InvalidViewTypeError
  • ImageRequestError
  • InvalidAdSizeError
  • InvalidInterstitialAdType The adTypes in the AdbladeInterstitialViewController will be set to the default when this error is sent
