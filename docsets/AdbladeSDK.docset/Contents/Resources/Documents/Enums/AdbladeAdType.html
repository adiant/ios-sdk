AdbladeAdType

@objc public enum AdbladeAdType: Int

Different types of ads.

  • native:
    • 298x224 image
    • title
    • description
    • display name
  • mediumRectangle: 300x250
  • richMediaMediumRectangle: 300x250 Rich Media
  • leaderboard: 728x90
  • richMediaLeaderboard: 728x90 Rich Media
  • banner: 320x50
  • mediumBanner: 320x100
  • fullBanner: 468x60
  • slideshow - a collection of the other ad types that can be rotated through

If you do not specify ad ad type, one or more will be chosen for you based on the selected view. If this is the case, the 300x250 ones can be grouped together and the 728x90 ones can be grouped together. Only one ad will be returned, but it can be of any of those types.
