Protocols

The following protocols are available globally.

  • AdbladeView

    The protocol used for all Adblade views. This should not be implemented.

  • AdbladeDelegate

    This can be used if you want to track any events during the process. All methods are optional and you only have to implement what you need.

    class ViewController: UIViewController, AdbladeDelegate { //… adbladeView.delegate = self //…

    func didReceiveAd(view: AdbladeView) { //… } }
